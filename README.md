# 恶臭数字论证 · AstrBot 插件

将任意数字「恶臭论证」为由 **114514 相关因子** 构成的算术表达式。  
支持整数与小数，会自动拆分并构造只包含 `()+-*/`（以及必要的幂运算）的公式，用于复刻经典 *homo / 恶臭数字论证* 梗。

---

## 指令说明

### 数字论证

> 实际触发方式视你的机器人前缀配置而定，下面仅示例核心指令部分。

```text
homo 1919
恶臭论证 1919
```

示例输出：

```
114 = 11*4+5*14
514 = (1-1)/4+514
114.514 = (114514)/(-11/4+51/4)\*\*(11\*-4+51-4)
```

## ⚙️ 插件配置

escape_markdown（是否转义 Markdown）
- 默认：false
- 类型：bool
- 作用：是否对输出结果中的 * 做 Markdown 转义（替换为 \*），以避免在支持 Markdown 的前端中被误解析为斜体/粗体。
